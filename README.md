# Трекер вакансий Росины

Откройте `index.html` двойным кликом. Сервер и установка библиотек не нужны.

Чекбокс **«Подалась»** сохраняется в локальном хранилище браузера. Он не исчезает после обновления списка, пока вакансия сохраняет свой ID.

## Обновить вакансии

Из корня `D:\Rosina WORK` выполните:

```powershell
powershell -ExecutionPolicy Bypass -File .\job-tracker\update-jobs.ps1
```

Команда запускает Codex, проверяет текущие вакансии, переносит закрытые в архив, добавляет новые, запускает тесты и пересобирает `index.html`.

Проверить окружение без поиска:

```powershell
powershell -ExecutionPolicy Bypass -File .\job-tracker\update-jobs.ps1 -CheckOnly
```

Для обновления нужны авторизованный Codex CLI, Node.js и доступ в интернет.
