# Обновление вакансий выбранного профиля

Это утверждённая операция обслуживания. Используй значения `PROFILE_ID`, `CV_PATH` и `PROFILE_DIR`, добавленные в конце prompt.

1. Прочитай приватный CV только для анализа. Никогда не копируй в публичные файлы телефон, email, адрес, фотографию или полный текст CV.
2. Прочитай `<PROFILE_DIR>/profile.json`, `<PROFILE_DIR>/criteria.json` и `<PROFILE_DIR>/jobs.json`.
3. Выполни широкий поиск по LinkedIn, локальным job boards, европейским remote-доскам и прямым карьерным страницам.
4. Проверяй географию, оформление, seniority и ограничения строго по `criteria.json`.
5. Проверяй прямую страницу работодателя, когда она доступна. Не добавляй дубли.
6. Новым вакансиям назначай стабильный ASCII slug ID. Существующие ID не меняй.
7. Закрытые вакансии не удаляй: ставь `status: "archived"`.
8. Все описательные поля пиши по-русски. Зарплату сохраняй дословно; если её нет — `"Не указана"`.
9. Вероятности должны быть числами 0..1; offerProbability не выше interviewProbability.
10. Изменяй только файлы выбранного `PROFILE_DIR`. Не меняй другие профили.
11. После обновления запусти тесты и генератор HTML.
